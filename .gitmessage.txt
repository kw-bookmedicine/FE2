#####[깃모지 및 커밋 유형 리스트]######
#  :sparkles: Feat: 새로운 기능 추가
#  :bug: Fix: 버그 수정
#  :memo: Docs: 문서 수정
#  :lipstick: Style: 내부 로직은 변경하지 않고 코드 스타일, 포맷 등을 수정한 경우, 줄 바꿈, 누락된 세미콜론 추가 등의 작업도 포함
#  :recycle: Refactor: 코드 리팩토링 (내부 로직은 변경하지 않고 기존의 코드를 개선한 경우, 클래스명 수정&가독성을 위해 변수명을 변경한 경우도 포함)
#  :package: Chore: 패키지 매니지 수정, 그 외 변경 사항 ex) .gitignore
#  :art: Design: CSS 등 사용자 UI 디자인 추가, 수정
#  :bulb: Comment: 필요한 주석 추가 및 변경 (❗ 필요 없는 주석을 삭제한 경우는 remove)
#  :truck: Rename: 파일 또는 폴더 명을 수정하거나 옮기는 작업인 경우
#  :fire: Remove: 파일을 삭제하는 작업만 수행한 경우
#  :label: Type: 타입을 추가하거나 수정하는 경우
#  :tada: Init: 초기 세팅 및 초기화
#  :white_check_mark: Test: 테스트 코드, 리팩토링 테스트 코드 추가
